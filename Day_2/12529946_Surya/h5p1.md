## Star pyramids
``` python
l = n
c = 2
for i in range(n):
    for j in range(l):
        print(" ",end = " ")
    l = l-1
    for k in range(1,c):
        print("*", end=" ")
    c = c+2
    print()
```
## Floyd Triangle
``` python
a = 1
l = []
i = 0
m=0
while i<100:
    ele = i+a
    l.append(ele)
    a = a+1
    i = ele
for q in range(1,16):
    print(q,end=" ")
    if q == l[m]:
        print()
        m = m+1
```
