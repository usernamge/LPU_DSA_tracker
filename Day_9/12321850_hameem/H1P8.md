# **Climbing Stairs**
# https://leetcode.com/problems/climbing-stairs/submissions/2053382222/
```python
class Solution:
    def climbStairs(self, n):
        if n <= 2:
            return n
        first = 1
        second = 2
        for i in range(3, n + 1):
            first, second = second, first + second
        return second
```
