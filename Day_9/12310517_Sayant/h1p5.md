# h1p5 Ugly Number

https://leetcode.com/problems/ugly-number/

```python

class Solution:
    def isUgly(self, n: int) -> bool:

        check=[1,2,3,5]

        for i in range(1,n):
            if n%i==0:
                if i not in check:
                    return False
        
        return True
```
