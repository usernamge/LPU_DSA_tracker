# h1p3 Happy Number

https://leetcode.com/problems/happy-number/

```python
class Solution:
    def isHappy(self, n: int) -> bool:
        x= str(n)
        num=[]
        check=[]

        while True:
            final=0
            for i in x:
                num.append(int(i))

            for i in num:
                final+=i**2
            
            if final==1:
                return True
            elif final in check:
                return False
            
            check.append(final)
            x=str(final)
            num=[]

```
