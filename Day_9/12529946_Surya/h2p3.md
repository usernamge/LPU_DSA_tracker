## Koko Eating Bananas : https://leetcode.com/problems/koko-eating-bananas/description/

```Python
import math

class Solution:
    def minEatingSpeed(self, piles: List[int], h: int):
        mn=1
        mx=max(piles)
        while mn<mx:
            i=(mn + mx)//2
            sum=0
            for j in piles:
                sum+= math.ceil(j/i)
            if sum<=h:
                mx=i
            else:
                mn=i+1

        return mn
```
