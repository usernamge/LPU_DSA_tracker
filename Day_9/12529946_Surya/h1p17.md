## Length of Last Word: https://leetcode.com/problems/length-of-last-word/description/
``` Python
class Solution:
    def lengthOfLastWord(self, s: str) -> int:
        count = 0
        s = s.rstrip()
        for i in reversed(s):
            if i.isalpha():
                count = count + 1
            else: break
        return count
```
