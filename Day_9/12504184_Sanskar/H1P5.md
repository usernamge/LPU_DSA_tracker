# 263 - Ugly Number

- https://leetcode.com/problems/ugly-number/description/

```python

class Solution:
    def isUgly(self, n: int) -> bool:
        if n <= 0:
            return False
        for prime in [2, 3, 5]:
            while n % prime == 0:
                n //= prime
        return n == 1   

```
