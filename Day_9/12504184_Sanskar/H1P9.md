# 7 - Reverse Integer

- https://leetcode.com/problems/reverse-integer/

```python

class Solution:
    def reverse(self, x: int) -> int:
        isNeg=1
        rev=0
        if(x<0): 
            isNeg=-1
            x*=-1
        while x:
            lastD = x%10
            rev = rev*10 + lastD
            x = x//10
        if rev > 2**31 - 1 or rev < -2**31:
            rev=0
        return rev*isNeg

```
