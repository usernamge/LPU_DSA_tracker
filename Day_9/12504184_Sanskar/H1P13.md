# 387 - First Unique Character in String


- https://leetcode.com/problems/first-unique-character-in-a-string/

```python

class Solution:
    def firstUniqChar(self, s: str) -> int:
        count = Counter(s)

        for i, c in enumerate(s):
            if count[c] == 1:
                return i
        return -1

```
