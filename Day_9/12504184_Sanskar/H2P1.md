# 704. Binary Search

https://leetcode.com/problems/binary-search/description/

```python

class Solution:
    def search(self, nums: List[int], target: int) -> int:
        l = 0
        r = len(nums)-1
        while l<=r:
            m = (l+r)//2
            if nums[m]==target:
                return m
            elif nums[m]>target:
                r = m-1
                m = (l+r)//2
            else:
                l = m+1
                m = (l+r)//2
        return -1 

```
