# 58. Length of Last Word

- https://leetcode.com/problems/length-of-last-word/description/

```python

class Solution:
    def lengthOfLastWord(self, s: str) -> int:
        count = 0
        end  = len(s)

        while end!=0:
            if s[end-1]==" ":
                end-=1
                if count!=0:
                    return count
                continue
            else:
                count+=1
                end-=1
        return count

```
