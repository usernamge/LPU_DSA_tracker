# 9 - Palindrome Number

- https://leetcode.com/problems/palindrome-number/

```python

class Solution:
    def isPalindrome(self, x: int) -> bool:
        s = str(x)
        p1=0
        p2=len(s)-1
        while p1<=p2:
            if s[p1]==s[p2]:
                p1+=1
                p2-=1
                continue
            
            else:
                return False
        
        return True

```
