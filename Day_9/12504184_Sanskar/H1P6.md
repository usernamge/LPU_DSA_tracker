# 69 - Sqrt(x)

- https://leetcode.com/problems/sqrtx/description/

```python

class Solution:
    def mySqrt(self, x: int) -> int:
        left = 0
        right = x

        middle = (left+right)//2

        while left<=right:
            if middle**2 == x:
                return middle
            elif middle**2 > x:
                right = middle-1
                middle = (left+right)//2
            elif middle**2 < x:
                left = middle+1
                middle = (left+right)//2

        return middle

```
