# 125 - Valid Palindrome

- https://leetcode.com/problems/valid-palindrome/

```python

class Solution:
    def isPalindrome(self, s: str) -> bool:
        cleaned = "".join(ch.lower() for ch in s if ch.isalnum())

        start = 0
        end = len(cleaned)-1

        while start<end:
            if cleaned[start]!=cleaned[end]:
                return False
            start+=1
            end-=1
        return True

```
