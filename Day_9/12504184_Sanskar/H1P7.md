# 66. Plus One

- https://leetcode.com/problems/plus-one/description/

```python

class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        d = len(digits)
        res=[]
        result = 0
        for i in range(len(digits)):
            result = result*10 + digits[i]
            d-=1
        
        result = str(result + 1)

        for i in range(len(result)):
            res.append(int(result[i]))
        
        return res

```
