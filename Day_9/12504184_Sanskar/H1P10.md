# 204 - Count Primes

- https://leetcode.com/problems/count-primes/

```python

class Solution:
    def countPrimes(self, n: int) -> int:
        if n<=2:
            return 0
        isPrime = [True]*n
        isPrime[0] = isPrime[1] = False
        for i in range(2, int(n**(0.5)+1)):
            if isPrime[i]:
                isPrime[i*i:n:i] = [False]*((n-1-i*i)//i+1)
        return sum(isPrime)

```
