# 70 - Climbing Stairs

- https://leetcode.com/problems/climbing-stairs/description/

```python

class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        prev2,prev1 = 1,2
        for _ in range(3, n + 1):
            current = prev1 + prev2
            prev2, prev1 = prev1, current
        return prev1

```
