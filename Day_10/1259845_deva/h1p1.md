# Reverse Linked List
https://leetcode.com/problems/reverse-linked-list/description/
```
python
class Solution:
    def reverseList(self, head):
        prev = None
        curr = head
        while curr:
            next_node = curr.next     
            curr.next = prev           
            prev = curr                
            curr = next_node          
        return prev
```
