## 344. Reverse String

- https://leetcode.com/problems/reverse-string/description/?envType=problem-list-v2&envId=string

```python
class Solution:
    def reverseString(self, s):

        start = 0
        end = len(s) - 1

        while start <= end:

            s[start], s[end] = s[end], s[start]

            start += 1
            end -= 1
```
