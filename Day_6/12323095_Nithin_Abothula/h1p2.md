# h1p2 Plus One

https://leetcode.com/problems/plus-one/submissions/2050341197/?envType=problem-list-v2&envId=array

```python

class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        for i in range(len(digits) - 1, -1, -1):
            if digits[i] < 9:
                digits[i] += 1
                return digits            
            digits[i] = 0            
        return [1] + digits

```
