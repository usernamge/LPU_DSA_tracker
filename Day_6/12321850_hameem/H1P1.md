# **REMOVE ELEMENT**

# https://leetcode.com/problems/remove-element/submissions/2051014943/?envType=problem-list-v2&envId=array
```python
class Solution(object):
    def removeElement(self, nums, val):
        k = 0
        for i in range(len(nums)):
            if nums[i]!=val:
                nums[k]=nums[i]
                k+=1
        return k
```
