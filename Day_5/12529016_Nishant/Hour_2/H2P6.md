# H2P6 - Count of Smaller Numbers After Self

- https://leetcode.com/problems/count-of-smaller-numbers-after-self/

```python
class Solution:
    def countSmaller(self, nums: List[int]) -> List[int]:
        counts = [0]*len(nums)
        for i in range(0,len(nums)):
            count = 0
            for j in range(i,len(nums)):
                if nums[j]<nums[i]:
                    count+=1
            counts[i]=count

        return counts
```
