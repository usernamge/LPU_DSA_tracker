### H3P6 — Jump Game (Greedy)

- https://leetcode.com/problems/jump-game/

```python
class Solution:
    def canJump(self, nums: list) -> bool:
        max_reach = 0
        for i, jump in enumerate(nums):
            if i > max_reach:
                return False
            max_reach = max(max_reach, i + jump)
        return True
```
