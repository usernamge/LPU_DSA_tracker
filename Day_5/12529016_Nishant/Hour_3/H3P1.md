### H3P1 — Climbing Stairs (Bottom-Up DP)

- https://leetcode.com/problems/climbing-stairs/

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        prev2, prev1 = 1, 2
        for _ in range(3, n + 1):
            curr = prev1 + prev2
            prev2 = prev1
            prev1 = curr
        return prev1
```
