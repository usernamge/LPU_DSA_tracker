### H5P3 — Number of Good Pairs

- https://leetcode.com/problems/number-of-good-pairs/

```python
class Solution:
    def numIdenticalPairs(self, nums: list) -> int:
        count = {}
        good_pairs = 0

        for num in nums:
            if num in count:
                good_pairs += count[num]
            count[num] = count.get(num, 0) + 1

        return good_pairs
```
