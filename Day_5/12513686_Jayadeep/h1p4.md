# Palindrome linked list 
- 
https://leetcode.com/problems/palindrome-linked-list/description/

'''python 

    class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        l = []
        while head is not None:
            l.append(head.val)
            head = head.next
        return l == l[::-1]

'''
